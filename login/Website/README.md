# Website Template

Eine saubere und einfache HTML/CSS/JavaScript-Vorlage für statische Websites.

## Struktur

- `index.html` - Hauptseite
- `subpage_1/` - Unterseite 1
- `subpage_2/` - Unterseite 2
- `subpage_3/` - Unterseite 3
- `res/` - Ressourcen (Bilder, Styles, Scripts)

## Verwendung

Passen Sie die Dateien an und fügen Sie Ihren Inhalt ein.
